HTML5

  • 태그를 열면 반드시 닫는다(meta 태그 예외)
  • <태그명/>로 열고 닫고를 한번에 처리 가능
  • html 문서의 구조만 결정하고 문서를 만들고
    화면의 처리는 css를 통해 표현한다.
  • 화면의 표현을 처리하던 태그들은 사라지고,
    의미를 갖는 태그는 존재함
  • html태그 자체가 표현은 무시하고 css로 재정의

HTML5 문법

h태그

  • h1~h6까지 있음
  • h의 특성은 제목을 적는것
  • 따라서 대제목으로 정한 한가지는 문서에 한번밖에 올 수 없음
    ex) h1이 대제목이면 h2부터 사용해야함 // h3가 대제목이면 h4부터 사용
  • 숫자가 연속되고 최상위는 단 하나
